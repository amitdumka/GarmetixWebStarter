# Print & Permission Final Acceptance v4.9.1

## Print acceptance

Open **Reports → Print Final Acceptance**.

For each document:

1. Open sample.
2. Verify logo/company details.
3. Verify date is correct.
4. Verify amount/tax/total matches screen value.
5. Verify signature blocks and page size.
6. Tick accepted.

## Permission acceptance

Open **Admin → Permission Final Acceptance**.

Test real login for:

- Admin / Owner
- Store Manager
- Billing user
- Purchase user
- Accountant
- HR/Payroll user

Verify scoped users cannot see other company/store records and non-admin users cannot access Data/Maintenance/Backup/Restore/Factory Reset areas.
