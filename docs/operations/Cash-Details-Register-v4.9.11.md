# Cash Details Register v4.9.11

Open:

```text
Accounting → Cash Details
```

## Add manual cash detail

1. Select store from top bar.
2. Open Cash Details.
3. Enter date, source and note/coin counts.
4. Click **Add cash detail**.

## Edit linked Day Opening / Day Closing cash

Linked rows show badge:

```text
Opening
Closing
```

Editing them updates linked day opening/closing balances.

## Delete

Manual rows can be deleted.

Linked Day Opening/Closing rows cannot be deleted from Cash Details. Use:

```text
Store Day Open / Close → Reopen Day / Delete Close
```

for day-close correction.
