# Stage Notes

Stage notes and validation logs are grouped by major stage:

- `stage-3/` - product, billing, purchase, and partial-return UI.
- `stage-4/` - numbering, concurrency, stock operations, GST reports, consistency, and repair.
- `stage-5/` - backup, production hardening, stabilization, migrations, and seeding.
- `stage-6/` - onboarding, public versioning, compile fixes, and Non-GST reporting.
- `stage-7/` - Nuxt UI dashboard migration through Stage 7M.
- `stage-8/` - full UI audit, responsive register standardization, and later Stage 8 implementation packages.

New stage notes and validation logs belong in the matching stage folder.
