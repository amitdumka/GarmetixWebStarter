# Stage 8G Package 5 - GSTIN Provider Validation (v4.6.4)

This package adds a go-live validation workflow for the selected licensed GSTIN provider.

## Included

- GSTIN provider status API.
- Admin-only provider test API.
- Production Readiness GSTIN provider panel.
- Mac mini readiness script for GSTIN configuration checks.
- Documentation for configuring provider URL, API key/header and audit source name.

The application still supports local GSTIN format/state-code validation when the provider is disabled. Live legal/trade name and address verification requires configured paid provider credentials.
