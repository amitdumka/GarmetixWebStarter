# Bank Account Edit Hotfix v4.8.4 Verification

After deployment:

1. Open **Accounting → Bank Accounts**.
2. Edit one existing bank account.
3. Change only branch or account holder name.
4. Save.
5. Refresh the page.
6. Confirm there is no duplicate new row.
7. Confirm the existing row contains the updated value.

If duplicates were created before this hotfix, delete/merge those duplicate rows manually after confirming which one is linked to transactions.
