# Aadwika Fashion + Smart Menswear Merge v4.9.4

## Before apply

1. Take database backup.
2. Export portable JSON seeder.
3. Open **Admin → AF/SS Seeder**.
4. Click **Preview merge**.
5. Review affected table counts.

## Apply

Tick confirmation and click **Apply merge**.

## After apply

1. Run **Data → Data Consistency**.
2. Open Companies/Store Groups/Stores.
3. Confirm:
   - Aadwika Fashion exists.
   - Aadwika Fashion MBO exists.
   - Smart Menswear store is under Aadwika Fashion MBO.
   - Aadwika Fashion - Shalini is still separate.
4. Test login/permissions and reports.
