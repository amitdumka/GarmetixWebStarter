# Stage 8G Package 8 - Access, HR/Payroll and Purchase Stabilization (v4.6.7)

This package addresses final go-live acceptance fixes reported during production testing.

## Included

- Legacy Overview is restricted to admin/owner only.
- Payroll route is available to store managers, power users and accounting users.
- Salary Structures tab is hidden from store/normal users and remains available to admin, owner, power user, accountant, remote accountant and payroll roles.
- Attendance now has an explicit Add Attendance action.
- Password reset now returns a clear production message when SMTP is not configured.
- Purchase New Inward is available as a full page at `/purchase/new`.
- Vendor Payments page added for invoice-linked and advance vendor payments with a payment register.
