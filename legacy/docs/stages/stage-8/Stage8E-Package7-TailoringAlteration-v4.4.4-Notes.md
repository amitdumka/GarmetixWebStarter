# Stage 8E Package 7 - Tailoring and Alteration Workflow (v4.4.4)

Build code: `GARMETIX-8E-20260617-4440`

Implemented before Stage 8F because garment retail operations need stitching and alteration handling.

## Scope

- Tailoring/stitching order register.
- Readymade garment alteration workflow.
- Multiple tailoring vendors using Vendor type `Tailoring` or `InHouse`.
- Reusable tailoring service items.
- Delivery schedule and overdue tracking.
- Customer receipts.
- Vendor payments.
- Order-to-service-invoice conversion.
- Order history/audit timeline.
- In-house alteration expense and profit impact tracking.

## SOP

Order -> Delivery -> Service Invoice -> Customer Payment.

Vendor payment can be recorded against the same tailoring order.
